# 首页使用路径

本文补充说明 Message Choose 首页的信息层级和推荐使用顺序。

1. 先看首页顶部“今日最该看”，确认今天最值得优先阅读的主事件和次热点。
2. 如果想追踪背景，进入“重点事件”页查看“发生了什么、为什么重要、接下来关注、证据条目”。
3. 如果已有明确主题，直接使用首页大搜索框；搜索结果会提示标题、关键词、摘要/来源等命中原因。
4. 搜索结果过少时，优先降低评分门槛或查看全部，再按频道和来源筛选。

设计原则：

- 热点优先展示事件，不只展示单条高分新闻。
- 解释优先展示事件意义和后续观察点。
- 移动端优先保留搜索框和主热点，次要统计与来源状态下沉。
